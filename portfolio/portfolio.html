Digital Twin of Ansaldo T100 Micro Gas Turbine

Developed with: Python

This digital twin application simulates the operation of the Ansaldo T100 Micro Gas Turbine. It provides two modes of operation: single-operation mode and dynamic simulation mode.

  • Single-Operation Mode: Allows users to set ambient conditions, desired power, and maximum error. Outputs include operational data for each point and maps for the compressor and turbine.
  • Dynamic Simulation Mode: Real-time operation tracking, including fuel flow rate, efficiency, and power output. Users can change the power setpoint online and observe changes in real-time.
